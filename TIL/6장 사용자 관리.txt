1. 리눅스 시스템에서 사용자 계정명, UID, GID, 홈 디렉터리, 셸의 종류 등 사용자 계정에 대한 자세한 정보가 저장되는 파일은?
- /etc/passwd

2. 리눅스 사용자의 암호화된 비밀번호가 저장되어 있는 파일은?
- /etc/shadow

3. sudo 명령에 관한 설명으로 잘못된 것은?
- sudo 명령을 사용할 때 사용자는 root 사용자의 비밀번호를 알고 있어야 한다.
- /etc/shdoers 파일에는 sudo 명령을 사용할 수 있는 사용자와 그 사용자에게 허용된 명령이 기록되어 있다.

4. 사용자 계정 생성과 관련이 있는 파일이 아닌 것은?
- /etc/fstab
- /etc/fstab 파일에는 부팅 시 자동으로 마운트되는 파일 시스템에 대한 정보가 저장되어 있다.

5. useradd 명령으로 사용자 계정을 추가하면 /home 디렉터리에 기본 사용자 디렉터리가 생성되고,
환경 설정을 위한 기본 파일들이 복사된다. 다음 중 사용자 기본 환경 설정 파일을 갖는 디렉터리는 무엇인가?
- /etc/skel

6. 사용자 계정을 실제로 삭제하지 않고 계정을 잠그는 명령은?
- usermod -L user

7. root 사용자가 사용자 kdhong의 비밀번호 변경에 사용하는 명령은?
- passwd kdhong

8. 사용자 계정과 그룹에 관한 일반적 설명
- 사용자 계정을 생성할 때 기본적으로 사용자 계정과 같은 이름의 그룹(GID도 UID와 같음)이 만들어진다.
- 사용자 계정은 0개 이상의 부 그룹에 속할 수 있다.
- 모든 사용자는 스스로 다른 부 그룹의 구성원으로 가입할 수 있다. -> X 사용자 계정을 특정 그룹의 구성원이 되게 하는 작업은 해당 그룹의 비밀번호
를 알고 있거나 관리 권한이 있어야만 가능하다.
- 모든 사용자 계정은 주 그룹을 가지며 그것의 구성원이 된다.
